OrangeHRM Manual Testing Project
A comprehensive Manual Testing Project for the OrangeHRM Web Application, covering end-to-end functional testing, test documentation, test execution, and defect reporting.
This project demonstrates clear understanding of STLC, testing techniques, test design, and real-time workflow followed in QA teams.
________________________________________
Project Overview
OrangeHRM is a popular HR Management application used for:
•	Login
•	Personal information management(PIM)
•	My Info
•	Leave Management
•	Time Tracking
•	Recruitment
•	Admin User Management
This project includes full manual test coverage for the critical modules of OrangeHRM.
________________________________________
 Scope of Testing
The following modules were covered in this project:
•	Login Module
•	Admin → User Management
Leave Module
•	PIM (Employee Information Management)
•	Recruitment Module
•	My Info Module
•	 Logout
Both positive and negative test scenarios were prepared.
________________________________________
Project Deliverables
This repository contains the complete set of manual testing documents:
1. Test Cases
Detailed, step-by-step test cases with:
•	Test Case ID
•	Objective
•	Pre-conditions
•	Test Steps
•	Expected Result
•	Priority
•	Test Data
2. Test Execution Report
Contains:
•	Execution status (Pass/Fail/Blocked)
•	Evidence screenshots
•	Retesting logs
 3. Bug Report
All defects logged with:
•	Bug ID
•	Severity & Priority
•	Steps to reproduce
•	Actual vs Expected Result
•	Screenshots
•	Status (New, Open, Fixed, Retested, Closed)
